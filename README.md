# CSharp-Console-Project-Lists-And-Collections

A simple C# console application developed using .NET 10.0.  
This project demonstrates the use of Lists and Collections by storing and displaying a collection of items in the console.

---

## 📌 Project Overview

The application creates a list of Apple Mac products and displays them using a `foreach` loop.

It demonstrates:

- Working with `List<T>` in C#
- Adding items to a collection
- Iterating through a list using `foreach`
- Basic console output handling

---

## 🛠 Technologies Used

- C#
- .NET 10.0 SDK
